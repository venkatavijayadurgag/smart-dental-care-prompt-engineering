# Prompt Versions – Task 2

## Prompt Version 1 (Original)

You are a professional website copywriter.

Business Type: Local Dental Clinic  
Target Audience: Families and working professionals  
Location: Hyderabad  

Create website copy with the following sections:
1. Homepage introduction with a clear value proposition
2. About section focused on trust and expertise
3. Services page descriptions for:
   - Dental Checkup
   - Braces
   - Teeth Whitening
4. Strong call-to-action section encouraging appointment booking

Tone: Friendly, professional, and trustworthy  
Language: Simple, clear, and website-ready
## Prompt Version 2 (Improved)

You are an expert website copywriter specializing in healthcare businesses.

Business Type: Local Dental Clinic  
Target Audience: Families and working professionals  
Location: Hyderabad  

Objective: Create high-converting, trust-building website copy that encourages appointment bookings.

Instructions:
- Write clear, benefit-focused content
- Highlight patient comfort, safety, and professional expertise
- Use short paragraphs and simple language
- Maintain a friendly, professional, and trustworthy tone

Required Sections:
1. Homepage introduction with a strong value proposition
2. About section emphasizing experience, modern equipment, and patient trust
3. Services section with clear descriptions for:
   - Dental Checkup
   - Braces
   - Teeth Whitening
4. Final call-to-action motivating users to book an appointment

Output Format:
- Use headings for each section
- Make the content website-ready
- Avoid unnecessary jargon
