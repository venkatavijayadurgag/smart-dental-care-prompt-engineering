You are a professional website copywriter.

Business Type: Local Dental Clinic  
Target Audience: Families and working professionals  
Location: Hyderabad  

Create website copy with the following sections:

1. Homepage introduction with a clear value proposition
2. About section focused on trust, expertise, and modern dental equipment
3. Services page descriptions for:
   - Teeth Cleaning
   - Dental Implants
   - Pediatric Dentistry
   - Smile Makeover
4. Strong call-to-action section encouraging appointment booking

Language: Simple, clear, and website-ready

## Prompt Version 2 (Improved)

You are an expert website copywriter specializing in healthcare and dental businesses.

Business Type: Local Dental Clinic  
Target Audience: Families and working professionals  
Location: Hyderabad  

Objective: Create high-converting, trust-building website copy that encourages appointment bookings and highlights realistic dental services.

Instructions:

- Write clear, benefit-focused content for each service
- Highlight patient comfort, safety, and professional expertise
- Use short paragraphs and simple language
- Maintain a friendly, professional, and trustworthy tone

Required Sections:

1. Homepage introduction with a strong value proposition
2. About section emphasizing experience, modern equipment, patient safety, and trust
3. Services section with detailed, realistic descriptions for:
   - Teeth Cleaning
   - Dental Implants
   - Pediatric Dentistry
   - Smile Makeover
4. Final call-to-action motivating users to book an appointment

Output Format:

- Use headings for each section
- Ensure content is website-ready
- Avoid unnecessary jargon
- Content should feel like a real, professional dental clinic website
